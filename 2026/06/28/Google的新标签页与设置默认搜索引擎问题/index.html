Google的新标签页与设置默认搜索引擎问题
发布于：2026-06-28 17:49:10 标签：/ askAI / 访问：

问题：

Google Chrome浏览器，在设置打开新的标签页时，如果默认搜索引擎是百度，则新标签页的内容显示的是8个历史访问记录图标，但如果默认搜索引擎是Bing时却是Bing的主页，这是为什么？

回答：

我明白你说的这个现象。这其实是 Chrome 浏览器一个特殊的设计逻辑：当默认搜索引擎是 Google（或百度这类使用自家新标签页的引擎）时，新标签页会显示一个导航快捷方式页面；而当换成 Bing 时，Chrome 会倾向于直接加载 Bing 的搜索主页。

🤔 为什么会这样？

这背后涉及到 Chrome 对“新标签页”和“搜索引擎”的整合策略，以及厂商之间不同的合作方式。

  1. “新标签页”与“搜索引擎主页”的分离：在 Chrome 的设计里，“新标签页”的 URL（地址是 chrome://newtab）和“搜索引擎主页”（如 bing.com）原本是两个概念。新标签页通常由 Chrome 自己控制，用来展示搜索框和常用网站。当你把默认搜索引擎设置为百度时，Chrome 保留了它的“新标签页”控制权，只把地址栏的搜索功能交给了百度，所以看到的还是那个带历史记录的快捷方式页。

  2. Bing 与 Chrome 的特殊整合：而当把搜索引擎设置为 Bing 时，情况就不同了。Chrome 和微软之间有合作，会把 Bing 的主页直接“嵌入”到新标签页里来显示。也就是说，当你新建标签页时，实际上加载的是 Bing 针对 Chrome 定制的新标签页版本（可能类似 https://www.bing.com/chrome/newtab），看起来就和 Bing 主页一样了。

简单来说，这种差异源于不同的合作策略：对于百度，Chrome 选择保留了自家新标签页的样式；对于 Bing，则采用了更深度整合的嵌入式方案。

💡 如何解决？

如果你更喜欢 Chrome 原生的新标签页，但还想用 Bing 来搜索，可以通过“曲线救国”的方式实现：

在 Chrome 设置里，进入“搜索引擎”->“管理搜索引擎和网站搜索”。

在“网站搜索”部分，点击“添加”按钮。

填写一个新的搜索引擎条目，关键是将 “URL” 设置为：https://www.bing.com/search?q=%s。

将你新添加的这个搜索引擎设为默认值。

完成这一步后，默认搜索引擎虽然还是 Bing，但因为它不再是被 Chrome 内置集成的那个 Bing 选项，新标签页就会恢复成 Chrome 默认的样式了。
