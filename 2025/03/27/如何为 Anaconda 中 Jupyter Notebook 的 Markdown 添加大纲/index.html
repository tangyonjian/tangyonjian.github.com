如何为 Anaconda 中 Jupyter Notebook 的 Markdown 添加大纲
发布于：2025-03-27 22:06:59 标签：/ Anaconda Jupyter Notebook / 访问：

1. 打开 Anaconda Powershell Prompt，如图：

2. 输入命令安装 jupyter_contrib_nbextensions

3. 安装 Jupyter 插件

4. 在 Anaconda 中启动 Jupyter，启动之后可以看到 Nbextensions 已安装

5. 点击 Nbextensions，取消勾选 disable configuration for nbextensions，并勾选 Table of Contents (2)

6. 新建或打开一个 Jupyter Notebook 文档，点击“Table of Contents”按钮，就可以查看大纲列表了

7. 点击“ToC settings”按钮，可以进行相关设置，如，是否自动编号，是否添加到文档本身等，如图：
